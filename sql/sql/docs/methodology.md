# Metodologia (visão geral)

- Dados: população (World Bank), prevalência de obesidade (OMS/GBD), critérios de elegibilidade (FDA), premissas de adesão/continuidade.
- Modelagem: dimensões (país, cenário) + fatos (população, obesidade, capacidade, preço).
- Cálculos: demanda (doses/ano), capacidade (doses/ano), % atendido, receita.
- Saída: queries SQL + dashboard Power BI.
