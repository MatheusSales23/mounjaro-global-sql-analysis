# Escopo do Projeto

Objetivo: estimar demanda global, capacidade de produção e potencial de receita da tirzepatida (Mounjaro/Zepbound) usando SQL Server e Power BI.

Perguntas:
- Quantas pessoas elegíveis existem por país/ano?
- Quantas doses/ano seriam necessárias em diferentes cenários?
- Em que cenário a oferta alcança a demanda?
- Qual a receita anual estimada por cenário?
